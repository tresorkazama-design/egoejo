## 1. Résumé Exécutif

**EGOEJO** est une plateforme numérique dédiée à la régénération du vivant qui génère un impact mesurable en reliant des citoyens à des projets sociaux à fort impact écologique et social. La plateforme combine un modèle de gouvernance innovant (double structure relationnelle/instrumentale) avec des mécanismes de transparence et d'auditabilité permettant une mesure d'impact rigoureuse.

### Résultats Clés (Année 1)

| Indicateur | Objectif | Mesure |
|------------|----------|--------|
| **Projets référencés** | 50-100 | À mesurer trimestriellement |
| **Utilisateurs actifs** | 500-1000 | À mesurer mensuellement |
| **Montant collecté** | 50 000-100 000 € | À mesurer mensuellement |
| **Taux de réussite collectes** | 60% | À mesurer trimestriellement |
| **Taux de participation** | 40% (≥3 projets) | À mesurer trimestriellement |

### Impact Attendu (3 Ans)

- **Court terme (Année 1)** : 50-100 projets, 500-1000 utilisateurs, 50 000-100 000 € collectés
- **Moyen terme (Année 2-3)** : 200-500 projets, 5000-10000 utilisateurs, 500 000-1 000 000 € collectés
- **Long terme (Année 4+)** : Modèle réplicable, label adopté par 10-20 projets tiers, impact environnemental mesurable

### Retour sur Investissement (ROI)

- **Ratio d'impact** : 1 € investi → 2-5 € collectés pour les projets (estimation)
- **Multiplicateur d'impact** : Chaque projet financé génère un impact environnemental et social mesurable
- **Soutenabilité** : Modèle non-lucratif, transparence totale, indépendance financière

## 4. Modèle de Gouvernance

### Double Structure (Protection Juridique & Technique)

EGOEJO opère selon une **double structure** strictement séparée et protégée :

#### Structure Relationnelle (SAKA)

- **Nature** : Monnaie interne d'engagement, non-financière, non-monétaire
- **Protection juridique** : Non-financier (réglementation AMF non applicable)
- **Protection technique** : Aucune conversion SAKA ↔ EUR possible (encodé dans le code, testé automatiquement)
- **Mécanisme anti-accumulation** : Compostage obligatoire, redistribution équitable

#### Structure Instrumentale (EUR)

- **Nature** : Monnaie réelle, instrumentale
- **Protection juridique** : Conformité via Stripe (agrément PSP, conformité DSP2)
- **Transparence** : 100% des dons nets (après frais) utilisés pour les projets
- **Traçabilité** : Audit logs, rapports trimestriels

### Subordination (Principe Fondamental)

**Principe** : La structure relationnelle (SAKA) **prime toujours** sur la structure instrumentale (EUR).

**Justification** :
- **Juridique** : Protection contre réglementation financière (AMF, DSP2)
- **Éthique** : Engagement relationnel prime sur accumulation matérielle
- **Technique** : Aucune conversion possible, protégée par le code

### Garde-Fous (Mesurables)

#### Techniques (Quantifiables)

1. **Tests automatiques** : 100% des tests de compliance passent (mesure : CI/CD)
2. **CI/CD bloquante** : 0 violation détectée (mesure : logs GitHub Actions)
3. **Audit logs** : 100% des actions critiques tracées (mesure : compteur logs)
4. **Endpoints publics** : 100% disponibilité (mesure : monitoring)

#### Juridiques (Vérifiables)

1. **Constitution technique** : Document fondateur non négociable (vérifiable)
2. **Label public** : Certification publique vérifiable (endpoint JSON)
3. **Gouvernance protectrice** : Conseil d'administration, review obligatoire (procès-verbaux)

#### Humains (Mesurables)

1. **Formation équipe** : 100% de l'équipe formée (mesure : attestations)
2. **Review obligatoire** : 100% des modifications critiques reviewées (mesure : logs)
3. **Documentation** : 100% de la documentation à jour (mesure : dernière mise à jour)
