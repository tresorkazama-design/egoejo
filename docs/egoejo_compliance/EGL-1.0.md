## 5. Governance Compliance Requirements

**5.1 Mandatory Requirements**

All Works and Derivative Works licensed under EGL-1.0 must comply with the following mandatory requirements:

**(a) Absence of Financial Language**: The Work must not contain any financial language, including but not limited to:
- Return on investment (ROI)
- Profit, profitability, rentability
- Financial gain, dividend, yield
- Financial performance, valuation
- Any language suggesting monetary value or financial return

**(b) Absence of Monetary Symbols**: The Work must not contain monetary symbols (€, $, EUR, USD, GBP) except in strictly informative, non-promotional contexts.

**(c) Absence of Financial Promises**: The Work must not contain any promises of financial return, guarantee, or assurance of monetary value.

**(d) Valid Publication Workflow**: If the Work is published, it must have followed the valid publication workflow (draft → pending → published) as defined in the Editorial Constitution.

**(e) Source Attribution**: The Work must have an identifiable source (URL, file, bibliographic reference, or structured source field).

**(f) License Attribution**: The Work must have an explicit license (Creative Commons, Public Domain, or documented specific license).

**5.2 Recommended Requirements**

The following requirements are recommended but not mandatory:

**(a) Audit Trail**: The Work should have an audit log recording creation, submission, publication, and modifications.

**(b) Publication Traceability**: The Work should record the author of publication, date of publication, and publication history.
