# Clause Anti-Capture - EGOEJO Compliance

## Principe

> **"EGOEJO n'est pas une marque. C'est une contrainte volontaire."**

Le badge EGOEJO Compliant atteste d'une **contrainte volontaire** vérifiée automatiquement, pas d'une certification officielle ou d'une marque déposée.

---

## Interdiction d'usage marketing abusif

### Usages interdits

Le badge EGOEJO Compliant **NE PEUT PAS** être utilisé pour :

1. **Promettre un rendement financier**
   - ❌ "Investissez dans notre projet, il est EGOEJO Compliant"
   - ❌ "Notre projet EGOEJO Compliant génère des rendements"
   - ❌ "EGOEJO Compliant = garantie de profit"

2. **Garantir une performance économique**
   - ❌ "Projet EGOEJO Compliant = performance garantie"
   - ❌ "EGOEJO Compliant = meilleur ROI"
   - ❌ "Badge EGOEJO = succès économique assuré"

3. **Impliquer une valeur monétaire du SAKA**
   - ❌ "1 SAKA = X EUR"
   - ❌ "SAKA convertible en euros"
   - ❌ "SAKA a une valeur monétaire"

4. **Faire de la publicité mensongère**
   - ❌ "Certifié EGOEJO" (le badge n'est pas une certification)
   - ❌ "Approuvé par EGOEJO" (le badge est automatique, pas une approbation)
   - ❌ "Partenaire officiel EGOEJO" (le badge n'implique pas de partenariat)

### Usages autorisés

Le badge EGOEJO Compliant **PEUT** être utilisé pour :

1. **Attester d'une contrainte volontaire**
   - ✅ "Ce projet respecte les principes EGOEJO"
   - ✅ "Ce projet est conforme aux règles EGOEJO"
   - ✅ "Ce projet a choisi la contrainte EGOEJO"

2. **Documenter une philosophie**
   - ✅ "Ce projet partage la philosophie EGOEJO"
   - ✅ "Ce projet applique les principes EGOEJO"
   - ✅ "Ce projet respecte la constitution EGOEJO"

3. **Montrer la transparence**
   - ✅ "Nos règles sont vérifiables automatiquement"
   - ✅ "Nos principes sont encodés dans le code"
   - ✅ "Notre conformité est traçable"

---

## Badge révocable

### Conditions de révocation

Le badge peut être **révoqué** si :

1. **Usage marketing abusif détecté**
   - Promesse de rendement financier
   - Garantie de performance économique
   - Publicité mensongère

2. **Violation des principes EGOEJO**
   - Conversion SAKA ↔ EUR
   - Rendement financier basé sur SAKA
   - Cycle SAKA contourné

3. **Tests de conformité non publics**
   - Tests cachés ou privés
   - Résultats non traçables
   - Refus de partager les tests

4. **Refus de corriger les violations**
   - Violations détectées non corrigées
   - Refus de répondre aux questions
   - Non-respect des règles

### Processus de révocation

1. **Signalement** : Ouvrir une issue sur le repo EGOEJO avec preuves
2. **Examen** : La communauté EGOEJO examine la demande
3. **Décision** : Révocation ou maintien du badge
4. **Notification** : Le projet est notifié de la décision

---

## Badge basé sur tests publics

### Règle absolue

Le badge n'est **valide** que si :

1. ✅ **Tests de conformité publics**
   - Les tests sont dans le repo public
   - Les tests sont exécutables par tous
   - Les résultats sont traçables

2. ✅ **Tests passent en CI**
   - Les tests sont exécutés automatiquement
   - Les résultats sont visibles publiquement
   - Les violations sont détectées automatiquement

3. ✅ **Résultats traçables**
   - Historique des vérifications disponible
   - Logs de CI accessibles
   - Rapport de conformité public

### Vérification

Toute personne peut :
- ✅ Exécuter `python egoejo-compliance/guardian.py`
- ✅ Obtenir le même résultat
- ✅ Vérifier la conformité indépendamment

## Exemples de violations

### Exemple 1 : Usage marketing abusif

**Violation** :
```
"Investissez dans notre projet EGOEJO Compliant !
Garantie de rendement de 10% par an."
```

**Action** : Révocation du badge + notification

### Exemple 2 : Publicité mensongère

**Violation** :
```
"Certifié EGOEJO - Approuvé par la plateforme EGOEJO"
```

**Action** : Correction demandée (le badge n'est pas une certification)

### Exemple 3 : Tests non publics

**Violation** :
- Tests de conformité dans un repo privé
- Résultats non accessibles publiquement

**Action** : Révocation du badge jusqu'à publication des tests
